medidas recomendadas para fontes == px e em

categorias de fontes == serifadas(com serifas), sans-serif( sem siifas),
monoespaçadas( todas as letras ocupam a mesma largura),
handwriting( simula letras escritas a mão),
display(fontes comemorativas)

variações de peso == lighter(leve), normal, bold, bolder
font-weigth == peso da fonte

========== comandos ===========
google fontes == site de fontes
dafont == site de fontes

@font-face == serve para usar uma fonte que você baixou
	 tipos de format()
		-  opentype (otf)
		- truetype (ttf)
		- embedded-opentype
		- truetype-aat (Appçe Advanced Typography)
		- svg

font-style:?; == estilo da fonte
font-weigth == peso da fonte
font-size == tamanho da fonte
font-family == familia da fonte
font: ; == shorthand onde você pode botar todas as configurações da fonte mais tem que ser em ordem
	font-style -> font-weigth -> font-size -> font-family
	ex: font: italic bolder 3em 'Worl Sans', sans-serif;

text-decoration:?; == decoração do texto

======== como identificar fontes ============
extenção no chrome para identificar fontes nos sites
	fonts ninja

sites para identificar fontes em imagens
	what font is, font squirrel, my fonts
=========alinhamento do texto ===========
text-align: justify; == alinha o texto para que uma linha termine no mesmo local que a outra
	right; alinha o texto a direita
	left; == alinha o texto a esqerda

text-indent: ?; == da uma identação no texto
